# Heens Restaurant Website

[Live Site](!https://heensrestaurant.uk/)

## Development

**Required**

- node
- npm / yarn
- gatsbyjs

**Clone repository**

```shell
yarn clone git@github.com:ThangTruongGit/heensrestaurant.git
```

**Install npm packages**

```shell
yarn install
```

or

```shell
npm install
```

**Run dev mode**

```shell
yarn dev
```
